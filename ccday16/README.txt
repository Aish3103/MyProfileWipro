This project contains the solutions for Day 16 of learning Node.js. It includes three small tasks to help understand Node.js fundamentals, npm usage, and creating a simple CLI application.

What This Project Includes:
1.A basic Node script that prints information and shows timed messages.
2.A simple demo using npm packages to display a styled welcome message.
3.A basic CLI tool that greets the user with their name and the current date and time.
4.A small configuration file (server.json).

How to Use

1.Install Node.js on your computer.
2.Run each file using the node command in the terminal.
3.Install any required npm packages with npm install.
4.Use npm start if a start script is included.

These tasks help you understand:
1.How Node.js works outside the browser
2.How npm manages dependencies
3.How to read input from the command line
